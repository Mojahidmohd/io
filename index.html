AWS Shopping Site Project

by Mojahid Mohd

📌 Project Overview

This project is a shopping website deployed on AWS using Terraform for infrastructure as code. It demonstrates a real-world cloud computing + DevOps use case.

🚀 Tech Stack

  • AWS EC2 – Web hosting for PHP/HTML site
  • AWS RDS (MySQL) – Backend shopping cart database
  • AWS ElastiCache (Redis) – Session & cart caching
  • AWS Cognito – User authentication
  • AWS API Gateway – Secure API calls
  • Terraform – Automated IaC deployment

📂 Features

  • Item catalog pulled dynamically from MySQL RDS
  • Add to cart & session persistence (via Redis)
  • User login/registration with Cognito
  • Infrastructure provisioned entirely with Terraform
